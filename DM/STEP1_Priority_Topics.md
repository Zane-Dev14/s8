## Answer Writing Rule (DM)

For 14-mark answers:
1. Definition
2. Key formula/idea
3. Step-by-step method
4. Small worked example
5. Advantages/limitations
6. Conclusion

For numericals:
1. Write given data
2. Write formula
3. Substitute values
4. Final boxed result

This structure gives method marks even with minor arithmetic slips.

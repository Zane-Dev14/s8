## How To Write For Marks

Part A (3 marks) writing format:
1. Write exactly 5 points.
2. Keep each point one short sentence.
3. Include formula in one point if numerical/theory needs it.
4. Include one tiny example or use-case.
5. End with one practical significance point.

Part B (14 marks) writing format:
1. Write exactly 10 points.
2. First 2 points = definition and core concept.
3. Middle 6 points = ordered explanation/steps.
4. Point 9 = example, table, or diagram instruction.
5. Point 10 = concluding insight.
